
《御定全唐詩》
===============

《全唐詩》，是清康熙四十四年（1705年），彭定求、沈三曾、楊中訥、汪士鋐、汪繹、俞梅、徐樹本、車鼎晉、潘從律、查嗣瑮等10人奉敕采錄，“得詩四萬八千九百餘首，凡二千二百餘人”，共計900卷，目錄12卷。 来自[百科](https://baike.baidu.com/item/%E5%85%A8%E5%94%90%E8%AF%97)
