
《御定全唐诗》
===============

《全唐诗》，是清康熙四十四年（1705年），彭定求、沈三曾、杨中讷、汪士𬭎、汪绎、俞梅、徐树本、车鼎晋、潘从律、查嗣瑮等10人奉敕采录，“得诗四万八千九百餘首，凡二千二百餘人”，共计900卷，目录12卷。 来自[百科](https://baike.baidu.com/item/%E5%85%A8%E5%94%90%E8%AF%97)
