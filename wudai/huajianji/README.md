花间集
=====

《花间集》是中国五代十国时期编纂的一部词集，也是文学史上的第一部文人词选集，由后蜀人赵崇祚编辑。本书收录了温庭筠、韦庄等18位花间词派诗人的经典作品，集中而典型地反映了早期词史上文人词创作的主体取向、审美情趣、体貌风格和艺术成就。


## 注意

本目录数据保存了所有的花间集数据诗词，由于五代十国朝代的原因， 可能与全唐诗和全宋词的诗词重复。  这是正常的。
