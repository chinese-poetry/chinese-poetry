南唐二主词
=========

《南唐二主词》，系南唐中主李璟、后主李煜撰。约成书于南宋，后世续有辑补，又有后人编写了各种版本

**注意：** 可能和全唐诗和全宋词内容有重复诗词发生， 这是正常的。
