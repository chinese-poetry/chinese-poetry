论语
====

中国儒家经典,《论语》是孔子及其弟子的语录结集，由孔子弟子及再传弟子编写而成，至战国前期成书。全书共20篇492章，以语录体为主，叙事体为辅，主要记录孔子及其弟子的言行，较为集中地体现了孔子的政治主张、伦理思想、道德观念及教育原则等。此书是儒家学派的经典著作之一，与《大学》《中庸》《孟子》并称“四书”，再加上《诗经》《尚书》《礼记》《周易》《春秋》，总称“四书五经”。 来自百科
