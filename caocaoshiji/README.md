# 曹操诗集

曹操喜欢用诗歌、散文来抒发自己政治抱负，反映民生疾苦，是魏晋文学的代表人物，其文学成就，
主要表当今诗歌上，散文也很有特点。曹操的诗歌，今存20多篇，全部是乐府诗体。
内容大体上可分三类。一类是关涉时事的，一类是以表述理想为主的，一类是游仙诗。

## 说明

本库收录操作诗词26首，部分诗词中标注了全网版本的疑字。

收录语言为简体中文。
